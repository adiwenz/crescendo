## How to Run Locally

### Prerequisites

* Flutter SDK (latest stable)
* Xcode (for iOS) or Android Studio
* A physical device recommended for audio input testing

### Setup

```bash
git clone https://github.com/your-username/crescendo.git
cd crescendo_mobile
flutter pub get
```

### Run

```bash
flutter run
```
